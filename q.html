sol-query -- search Solid and Linked Data resources

Query RDF files and SPARQL endpoints using triple patterns, CSS selectors, and SPARQL.
Display results as a table, a rolodex, an autocomplete field and other built-in and custom vies.

  • triple patterns
    <sol-query endpoint="foo.ttl" wanted="? schema:location Mumbai"></sol-query>
  • RDFa selectors
    <sol-query endpoint="foo.html" wanted="[property=schema:name]"></sol-query>
  • stored SPARQL
    <sol-query endpoint="/foo/bar" sparql="./myQueries.ttl#BazQuery"></sol-query>
  • inline SPARQL
    <sol-query endpoint="/foo/bar" sparql="SELECT ?name WHERE {?p schema:name ?name.}"></sol-query>
    • view="rolodex"
    • view="table"
    • view="auto-complete"
    • view="accordion"
