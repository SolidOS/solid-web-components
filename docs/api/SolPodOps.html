Class: SolPodOps

SolPodOps()

new SolPodOps()

Standalone pod file/folder operations panel.

Renders the same tabbed interface as the sol-pod gear-icon modal, but inline. Tabs: Live Edit, View, Edit, Graph, Download, Rename, Delete, Permissions (files); New File, New Folder, Download, Rename, Delete, Permissions (containers).
