solid-web-components

Access Solid and Linked Data resources using HTML custom elements

Invoking the components Customizing the CSS
sol-menu Define a menu entirely in RDF using the ui: vocabulary; run it as a GUI with a simple web component; run it in a terminal with a single command.
sol-query Search data sources using inline or stored SPARQL or using triple patterns. Display the results as a table, a rolodex, other built-in and custom views.
sol-login Creates a login button that opens a dropdown with a list issuers, then presents the user with a login screen for the selected issuer. The authenticated fetch is propagated to all other fetches, including Comunica queries and HTML/Markdown page includes.
sol-include Fetch text based documents and display their contents inline. Supports authenticated fetch, robust modular page construction, CSS/RDFa filetering, and always sanitizes by default.

Components for SHACL-validated ui: forms, login, and others coming soon!
