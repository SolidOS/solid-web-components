# Solid Components

**This is a Work In Progress - More Help Coming Soon!**

This repo contains two related packages - @solid-components/web, a set of web-components for Solid; and @solid-components/node, a nodejs version of some of the web-components for using in a script or terminal command line. 

Curent components: 
```
  + = done but not documented
  - = in progress
  x = not applicable

                 web  node 
   sol-menu       +    +    create menus from RDF or HTML
   sol-query      +    +    query using SPARQL or triple patterns
   sol-login      +    +    login to Solid
   sol-include    +    +    transclude documents with sanitizing
   sol-form       +    -    create solid-ui forms
   sol-solidos    +    x    display resources in SolidOS
   sol-pod        +    ?    expandable pod folder/file tree
   sol-pod-ops    +    ?    complete file & folder management
   sol-wac        +    ?    manage .acl files
   sol-live-edit  +    x    RDF & other editors with graphic display of graphs
   sol-accordion  +    x    GUI accordion
   sol-modal      +    x    GUI modal
   sol-rolodex    +    ?    GUI rolodex
```
