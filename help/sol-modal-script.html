sol-modal — Script API

Create and control modals entirely from JavaScript. The SolModal class is both a custom element and a utility — use it imperatively without ever writing a tag.

Creating Modals

import { SolModal } from './sol-modal.js'; const modal = document.createElement('sol-modal'); modal.modalTitle = 'My Dialog'; modal.tabs = [ { name: 'View', render(body, footer, actions) { ... } } ]; modal.open();

The modal auto-appends to document.body when open() is called and removes itself on close().

Tab Callbacks

Each tab is an object with a name and a render function. The render function receives three shadow DOM elements to populate:

ParameterTypeDescription
bodyElementMain content area. Cleared before each render.
footerElementBottom button row. Cleared before each render.
actionsElementHeader action area (next to the close button). Cleared before each render.

The render function may return a cleanup callback that is called when switching to another tab or closing the modal:

{
  name: 'Editor',
  render(body, footer, actions) {
    const editor = createEditor(body);

    const saveBtn = document.createElement('button');
    saveBtn.textContent = 'Save';
    saveBtn.className = 'modal-header-btn modal-header-btn-primary';
    footer.appendChild(saveBtn);

    // Cleanup when leaving this tab
    return () => editor.destroy();
  }
}
When there is only one tab, the tab bar is hidden. The single tab's render() is still called normally.

Methods

MethodDescription
open()Renders the modal shell, appends to document.body, and activates the first tab.
close()Calls any active cleanup, fires sol-close, calls onClose, and removes from DOM.
switchTab(name)Switches to the named tab (case-insensitive). Calls cleanup on the previous tab, then calls render() on the new one.

Properties

PropertyTypeDescription
modalTitlestringGet/set the title (reflects title attribute).
tabsArrayGet/set tab definitions. Setting re-renders the tab bar if the modal is open.
onCloseFunction|nullCallback invoked on close (in addition to the sol-close event).
bodyElementThe modal body container (read-only, available after open()).
footerElementThe modal footer container (read-only).
headerActionsElementThe header actions container (read-only).

SolModal.prompt()

const value = await SolModal.prompt(message, placeholder?);

Static method that opens a small modal with an input field. Returns a Promise that resolves to:

  • The trimmed input string if the user clicks OK or presses Enter
  • null if cancelled (Cancel button, Escape, backdrop click, or close button)
ParameterTypeDescription
messagestringDisplayed as the modal title.
placeholderstringInput placeholder text. Default: ''.

Examples

File viewer with tabs

import { SolModal } from './sol-modal.js';

const modal = document.createElement('sol-modal');
modal.modalTitle = 'profile.ttl';
modal.tabs = [
  {
    name: 'View',
    render(body) {
      const pre = document.createElement('pre');
      pre.textContent = turtleText;
      pre.style.cssText = 'padding:16px;margin:0;overflow:auto';
      body.appendChild(pre);
    }
  },
  {
    name: 'Edit',
    render(body, footer) {
      const ta = document.createElement('textarea');
      ta.value = turtleText;
      ta.style.cssText = 'width:100%;height:300px;font-family:monospace';
      body.style.padding = '16px';
      body.appendChild(ta);

      const btn = document.createElement('button');
      btn.textContent = 'Save';
      btn.className = 'modal-header-btn modal-header-btn-primary';
      btn.onclick = async () => {
        await saveTurtle(ta.value);
        modal.close();
      };
      footer.appendChild(btn);
    }
  }
];
modal.open();

Confirmation dialog

function confirm(message) {
  return new Promise(resolve => {
    const m = document.createElement('sol-modal');
    m.setAttribute('size', 'small');
    m.modalTitle = message;
    m.tabs = [{
      name: '_confirm',
      render(body, footer) {
        const yes = document.createElement('button');
        yes.textContent = 'Yes';
        yes.className = 'modal-header-btn modal-header-btn-primary';
        const no = document.createElement('button');
        no.textContent = 'No';
        no.className = 'modal-header-btn';

        footer.appendChild(no);
        footer.appendChild(yes);

        yes.onclick = () => { m.close(); resolve(true); };
        no.onclick = () => { m.close(); resolve(false); };
      }
    }];
    m.onClose = () => resolve(false);
    m.open();
  });
}

Header actions (download button)

const modal = document.createElement('sol-modal');
modal.modalTitle = 'image.png';
modal.tabs = [{
  name: 'View',
  render(body, footer, actions) {
    const img = document.createElement('img');
    img.src = imageUrl;
    img.style.maxWidth = '100%';
    body.appendChild(img);

    // Add download button to header
    const dl = document.createElement('button');
    dl.textContent = '\u2B07 Download';
    dl.className = 'modal-header-btn';
    dl.onclick = () => window.open(imageUrl);
    actions.appendChild(dl);
  }
}];
modal.open();
