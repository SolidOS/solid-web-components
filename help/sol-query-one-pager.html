sol-query -- search Solid and Linked Data resources

Query RDF documents and SPARQL endpoints using triple patterns, inline SPARQL strings, or stored SPARQL queries. Display results as a table, a rolodex, or other built-in or custom views. 

    endpoint - URI of an RDF document or SPARQL endpoint
    wanted - triple pattern
    sparql - SPARQL string, or URI of a stored query
    view - name of built-in view or URI of custom view (default:table)
inline SPARQL 
<sol-query
    endpoint="../data/sample-data.ttl"
    query="SELECT ?name WHERE {?x  ?name.}""
</sol-query>

stored SPARQL with a replaceable parameter 
<sol-query
    endpoint="../data/sample-data.ttl"
    query="./data/myQueries.ttl#MyReplaceableQuery"
    var-myLimit="3"
</sol-query>
  • view="rolodex" 
    <sol-query 
  endpoint="../data/sample-data.ttl"
  query="../data/myQueries.ttl#MyBasicQuery"
  view="rolodex"
></sol-query>
  • view="table"
  • view="auto-complete"
  • view="accordion"
  • view="dl" (description list)
