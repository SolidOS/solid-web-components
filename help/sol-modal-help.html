sol-modal : dialog with tabs

  • Generic modal dialog web component with optional tabbed interface
  • Content rendered via tab callbacks — fully dynamic
  • Built-in SolModal.prompt() for quick input dialogs
  • Closes on Escape, backdrop click, or close button
  • Shadow DOM with CSS custom properties for theming

1. Single-Tab Modal

Create a <sol-modal> element, set its title and a single tab. The tab bar is hidden when there is only one tab.

<sol-modal title="File Details"></sol-modal>

Content is added programmatically via the tab's render(body, footer, actions) callback (see Script API).

2. Multi-Tab Modal

With multiple tabs, a tab bar appears. Users click tabs to switch views.

<sol-modal title="config.ttl"></sol-modal>
<!-- tabs: View, Edit, Info — set via .tabs property -->

3. Quick Prompt

Use the static SolModal.prompt() method for a simple input dialog. Returns the entered value or null if cancelled.

<script type="module">
  import { SolModal } from './sol-modal.js';
  const name = await SolModal.prompt('Enter file name:', 'untitled.ttl');
  if (name) console.log('User entered:', name);
</script>
