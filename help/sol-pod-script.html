pod-ops Functions

The shared/pod-ops.js module exports low-level functions for working with Solid pods without the <sol-pod> UI. All functions that access the pod accept a fetchFn parameter — pass an authenticated fetch from AuthManager.fetchFor().

import { fetchContainer, copyFile, copyFolder, deleteFolder, getAclUrl, getPermissions, contentTypeFor, extOf, MIME_TYPES, } from './shared/pod-ops.js';

Container Operations

FunctionReturnsDescription
fetchContainer(url, fetchFn) Promise<Array> Lists container contents via LDP. Returns [{ name, url, isContainer, size, modified }].
copyFile(src, destContainer, name, srcFetch, destFetch) Promise<{success}> Copies a file from one location to another. Separate fetch functions for source and destination.
copyFolder(src, destContainer, name, onProgress, srcFetch, destFetch) Promise<{success, failed}> Recursively copies a folder. onProgress receives status messages.
deleteFolder(url, fetchFn) Promise Recursively deletes a container and all its contents.

ACL Utilities

Functions for working with WAC (Web Access Control) ACL files.

import { getAclUrl, getPermissions, parseAcl, authsToRoleModel, roleModelToTurtle, adaptInheritedAcl, ROLES, GRANT_OPTIONS, } from './shared/pod-ops.js';
FunctionReturnsDescription
getAclUrl(resourceUrl, fetchFn)Promise<string>Discovers the ACL URL via the Link header. Falls back to url + '.acl'.
getPermissions(url, fetchFn)Promise<Object>Returns { own, aclUrl, inherited, inheritedFrom }. Walks up containers to find inherited ACL.
parseAcl(turtle, baseUrl)ArrayParses ACL Turtle into authorization objects.
authsToRoleModel(auths)ObjectConverts authorizations to a role-based model (owner, editors, viewers, public, etc.).
roleModelToTurtle(model, resourceUrl)stringConverts a role model back to ACL Turtle.
adaptInheritedAcl(turtle, resourceUrl)stringAdapts an inherited ACL for use as a direct ACL on the given resource.

Constants

NameDescription
ROLESPredefined role names: owner, editor, viewer, etc.
GRANT_OPTIONSAvailable permission grants: Read, Write, Append, Control.

Discovery

import { discoverOwnerWebIds, getStoragesFromWebIds, } from './shared/pod-ops.js';
FunctionReturnsDescription
discoverOwnerWebIds(origin?) Promise<string[]> Fetches /.well-known/openid-configuration and the root / to discover owner WebIDs via solid:oidcIssuer and acl:owner triples. Defaults to window.location.origin.
getStoragesFromWebIds(webIds, fetchFn?) Promise<string[]> Fetches each WebID profile and collects pim:storage values. Also follows owl:sameAs links.

Examples

List a container

import { fetchContainer } from './shared/pod-ops.js';

const items = await fetchContainer('https://pod.example/public/', fetch);
for (const item of items) {
  console.log(item.isContainer ? '📁' : '📄', item.name, item.url);
}

List with authenticated fetch

import { fetchContainer } from './shared/pod-ops.js';
import { AuthManager } from './sol-login.js';

const auth = new AuthManager();
// ... login flow ...

const url = 'https://pod.example/private/';
const authFetch = auth.fetchFor(url);
const items = await fetchContainer(url, authFetch);

Copy a file between pods

import { copyFile } from './shared/pod-ops.js';

const srcFetch = auth.fetchFor(srcUrl, 'left');
const destFetch = auth.fetchFor(destUrl, 'right');
const result = await copyFile(
  'https://pod-a.example/file.ttl',
  'https://pod-b.example/backups/',
  'file.ttl',
  srcFetch,
  destFetch
);
console.log(result.success ? 'Copied!' : 'Failed');

Read and modify ACL

import { getPermissions, parseAcl, authsToRoleModel, roleModelToTurtle } from './shared/pod-ops.js';

const perms = await getPermissions('https://pod.example/file.ttl', authFetch);
if (perms.own) {
  const auths = parseAcl(perms.own, perms.aclUrl);
  const model = authsToRoleModel(auths);
  console.log('Roles:', model);

  // Modify and save back
  const newTurtle = roleModelToTurtle(model, 'https://pod.example/file.ttl');
  await authFetch(perms.aclUrl, {
    method: 'PUT',
    headers: { 'Content-Type': 'text/turtle' },
    body: newTurtle
  });
}

Discover pods on a server

import { discoverOwnerWebIds, getStoragesFromWebIds } from './shared/pod-ops.js';

const webIds = await discoverOwnerWebIds('https://solidcommunity.net');
const storages = await getStoragesFromWebIds(webIds);
console.log('Pods:', storages);
// e.g. ['https://solidcommunity.net/jeff-zucker/', ...]

Programmatic navigation

const pod = document.querySelector('sol-pod');

// Navigate to a subfolder
await pod.loadContainer('https://pod.example/documents/');

// Read current state
console.log('Path:', pod.currentPath);
console.log('Items:', pod.items.map(i => i.name));
