sol-live-edit

A live split-pane editor web component with CodeMirror syntax highlighting, preview rendering, and format-specific help.

Usage Attributes API Events Editors sol-query sol-include

Usage

Add the component to any page. The only requirement is loading the module:

<script type="module" src="sol-live-edit.js"></script>

<sol-live-edit format="turtle"></sol-live-edit>

To load content from a URL:

<sol-live-edit source="https://example.org/data.ttl"></sol-live-edit>

The format is auto-detected from the file extension or Content-Type header. You can override it with the format attribute.

Attributes

Attribute Type Description
source string URL to fetch content from. Supports authenticated fetch via fetchFn property.
format string One of: turtle, jsonld, csv, markdown, mermaid, html, graphviz. Auto-detected from source extension if omitted.
readonly boolean Disables the save button when present.

JavaScript API

Property / Method Description
.contentGet or set the editor text content
.formatCurrent format (read-only)
.fetchFnSet an authenticated fetch function (e.g. Solid session.fetch)
.save()PUT content back to the source URL
.zoomIn() / .zoomOut()Zoom preview (turtle, jsonld, graphviz)
.toggleSettings()Show/hide settings panel
.toggleHelp()Show/hide format-specific help overlay
.toggleStats()Show/hide CSV statistics panel

Events

Event Detail Fires when
sol-change{content}Editor content changes
sol-save{content, url}Save is triggered
sol-load{content, url}Content loaded from source URL
sol-format{format, canZoom, canStats}Format changes
sol-zoom{zoom, pct}Zoom level changes
